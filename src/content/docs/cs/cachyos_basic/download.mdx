## Stažení ISO

ISO obraz CachyOS lze získat z následujících zdrojů:

* [Webová stránka](<https://cachyos.org/download>)
* [SourceForge](<https://sourceforge.net/projects/cachyos-arch/files/>)
* [CachyOS Mirror](<https://mirror.cachyos.org/ISO/>)

## Ověření integrity ISO pomocí SHA256

:::caution[VAROVÁNÍ]
Vždy proveďte krok navíc k ověření integrity ISO, abyste předešli nežádoucím problémům při instalaci nebo vytváření spustitelného média.
:::

**Aktuální verze ISO:** `250422`

**SHA256 Hash:** `c87272fc0e8476ea187005d416fa0ee7630a50c71c097233960a0fee22cb8bfe`

<Tabs>

<TabItem label='Windows'>

<Steps>

1.  Stáhněte [soubor](https://mirror.cachyos.org/ISO/desktop/250422/cachyos-desktop-linux-250422.iso.sha256) obsahující SHA256 hash **(Otevřete jej v textovém editoru, např. Poznámkový blok).**
2.  Otevřete CMD nebo PowerShell jako správce a [přejděte](<https://www.wikihow.com/Change-Directories-in-Command-Prompt>) do adresáře, kde jsou uloženy soubory ISO a SHA256.
3.  Spusťte následující příkaz:
    ```powershell
    # Příklad:
    certUtil -hashfile cachyos-desktop-linux-250422.iso SHA256
    ```
4.  Porovnejte výstup hashe z certUtil s hashem ze staženého souboru v **kroku 1.** Pokud se shodují, můžete pokračovat v instalaci CachyOS.

</Steps>

</TabItem>

<TabItem label='Linux'>

<Steps>

1.  Stáhněte [soubor](https://mirror.cachyos.org/ISO/desktop/250422/cachyos-desktop-linux-250422.iso.sha256) obsahující SHA256 hash.
2.  Otevřete terminál, přejděte do adresáře obsahujícího soubor `.sha256` a spusťte následující příkazy:
    ```sh
    # Příklad:
    cd Downloads/
    cat cachyos-desktop-linux-250422.iso.sha256
    # c87272fc0e8476ea187005d416fa0ee7630a50c71c097233960a0fee22cb8bfe
    ```
3.  Porovnejte výstup z **kroku 2** a spusťte následující příkaz pro kontrolu aktuálního hashe v ISO souboru.
    ```sh
    # Příklad:
    sha256sum cachyos-desktop-linux-250422.iso
    ```
4.  Pokud se hashe z **kroku 2** a **kroku 3** shodují, můžete pokračovat v instalaci CachyOS.

</Steps>

</TabItem>

## Ověření pravosti obrazu ISO (Linux)

Pro ověření pravosti ISO souboru, abyste se ujistili, že byl skutečně vydán oficiálním vývojovým týmem CachyOS:

<Steps>

1.  Importujte GPG klíč pro ověření pravosti:

    ```shell
    gpg --keyserver hkps://keys.openpgp.org --recv-key F3B607488DB35A47
    ```

2.  Stáhněte ISO soubor a jeho soubor s podpisem `.sig` a spusťte následující příkaz (nahraďte `full_iso_name.iso` skutečným názvem ISO souboru):
    ```shell
    gpg --verify full_iso_name.iso.sig full_iso_name.iso
    ```

    Pokud obdržíte výstup **Good signature**, ISO soubor je pravý:
    ```
    gpg: Signature made Mo 01 Apr 2024 14:13:30 CEST
    gpg:                using RSA key 882DCFE48E2051D48E2562ABF3B607488DB35A47
    gpg: Good signature from "CachyOS <admin@cachyos.org>" [unknown]
    gpg: WARNING: This key is not certified with a trusted signature!
    gpg:          There is no indication that the signature belongs to the owner.
    Primary key fingerprint: 882D CFE4 8E20 51D4 8E25  62AB F3B6 0748 8DB3 5A47
    ```
    *(Poznámka: Výstup GPG zůstává v angličtině, protože je to přímý výstup programu)*

</Steps>

:::danger
Pokud výstup neobsahuje řetězec **Good signature** nebo ID klíče nesouhlasí, nepoužívejte obraz ISO a ujistěte se, že stažený obraz pochází z legitimního zdroje CachyOS. Špatný podpis může naznačovat, že se staženým obrazem bylo manipulováno.
:::

## FAQ

### Wie bekomme ich den gleichen Look wie in den KDE Screenshots, genannt "Emerald"?

- **Es ist eine Kombination aus dem Emerald Theme + Qogir Icon Theme, die nicht standardmäßig installiert sind.**
  - ```sh title='Beide mit folgendem Befehl installieren'
    sudo pacman -S cachyos-emerald-kde-theme-git qogir-icon-theme
    ```
  **Beide unter `Erscheinungsbild und Stil - Farben und Designs` anwenden.**

### Warum sieht SDDM so fade und alt aus?

- *Da standardmäßig kein SDDM-Theme angewendet wird, greift SDDM auf das klassische Theme zurück.*
  - **Das Breeze Theme wie im Screenshot gezeigt anwenden**

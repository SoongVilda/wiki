## Systemanforderungen

Bevor Sie mit den Installationsvorbereitungen beginnen, stellen Sie sicher, dass der verwendete Computer die erforderlichen Systemanforderungen erfüllt, um CachyOS auszuführen. Das Installationsprogramm
verwendet einen Online-Installationsprozess, daher ist eine stabile und relativ schnelle Internetverbindung erforderlich.

<Tabs>

<TabItem label="Mindestanforderungen">

*   3 GB RAM
*   30 GB Speicherplatz (HDD/SSD)
*   Stabile Internetverbindung

:::caution
Eine instabile Internetverbindung kann im schlimmsten Fall zu einer längeren Installationsdauer oder fehlgeschlagenen Installationen führen.
:::

</TabItem>

<TabItem label="Empfohlene Anforderungen">

*   8 GB RAM
*   50 GB Speicherplatz (SSD/NVMe)
*   [x86-64-v3-fähige CPU](/de/installation/installation_prepare#x86_64-mikroarchitektur-level-unterstützung)
*   50 Mbit/s oder bessere Internetgeschwindigkeit
*   NVIDIA-GPU (900+ - z. B. GTX 950), AMD +GCN 1.0 (z. B. AMD R7 240) oder Intel (integrierte HD Graphics-Serie oder höher. Arc-Serie)

</TabItem>

</Tabs>

## Erstellen eines bootfähigen CachyOS-USB-Laufwerks

Es gibt verschiedene Tools und Methoden, mit denen ein bootfähiges USB-Laufwerk von CachyOS erstellt werden kann.

:::note
Das USB-Laufwerk muss mindestens 8 GB freien Speicherplatz haben.
:::

### balenaEtcher

<Steps>

1. Laden Sie [balenaEtcher](<https://etcher.balena.io/>) herunter. balenaEtcher unterstützt das Flashen eines Images von Linux, MacOS und Windows.
   Wenn eine `.zip`-Datei heruntergeladen wurde, extrahieren Sie diese und führen Sie die Binärdatei mit den folgenden Befehlen aus:
      ```sh
      # Zip-Datei extrahieren:
      bsdtar xvf <Name der Zip-Datei>
      # cd zum neuen Ordner:
      cd balenaEtcher-linux-x64/
      # balenaEtcher ausführen:
      ./balena-etcher
      ```
2. Schließen Sie ein USB-Laufwerk an den Computer an.
3. Wählen Sie **`Flash from file`** und wählen Sie die CachyOS-ISO-Image-Datei aus.
4. Klicken Sie auf **`Select target`** und wählen Sie das zuvor angeschlossene USB-Laufwerk aus.
5. Drücken Sie **`Flash`**, um den Flash-Vorgang zu starten.

</Steps>

:::note
balenaEtcher fordert möglicherweise Administratorrechte an. Geben Sie in diesem Fall Ihr Passwort ein und klicken Sie auf **`Authentifizieren`**.
:::

### Befehlszeilenschnittstelle (Linux & MacOS)

<Steps>

1. Schließen Sie ein USB-Laufwerk an.
2. Erkennen Sie das angeschlossene USB-Laufwerk mit den folgenden Befehlen:
   ```sh
   # Linux
   # Das USB-Laufwerk könnte z. B. als /dev/sda anhand des Festplattenmodells und der Größe identifiziert werden.
   sudo fdisk -l

   # MacOS
   # USB-Laufwerke werden als "/dev/diskY" identifiziert, wobei "Y" 0, 1 usw. sein kann.
   diskutil list
   ```
3. Kopieren Sie den Inhalt der ISO-Datei auf das angeschlossene USB-Laufwerk, indem Sie Folgendes ausführen:
   ```sh
   # Ersetzen Sie <usbdrive> durch die Bezeichnung des Laufwerks.
   sudo dd bs=4M if=full_iso_name.iso of=/dev/<usbdrive> status=progress oflag=sync
   ```
</Steps>

- `dd` kopiert nun den Inhalt der ISO-Datei auf das angeschlossene USB-Laufwerk.

### Rufus (nur Windows)

<Steps>

1. Schließen Sie ein USB-Laufwerk an.
2. Laden Sie [Rufus](<https://rufus.ie/>) herunter und installieren Sie es, oder führen Sie die portable Version aus.
3. Klicken Sie unter **`Device`** auf die Dropdown-Liste und wählen Sie das angeschlossene USB-Laufwerk aus.
4. Klicken Sie unter **`Boot selection`** auf **`SELECT`** und suchen Sie die CachyOS-ISO-Image-Datei.
5. Klicken Sie auf **`START`**.

</Steps>

### Ventoy (Linux & Windows)

<Tabs>

<TabItem label="Linux">

<Steps>

1. Schließen Sie ein USB-Laufwerk an.
2. Laden Sie [Ventoy](<https://www.ventoy.net/en/download.html>) herunter.
3. Entpacken Sie Ventoy mit dem folgenden Befehl:
   ```sh
   # Ersetzen Sie die Version durch die richtige, z. B. ventoy-1.0.99-linux.tar.gz
   bsdtar xvf ventoy-version-linux.tar.gz
   # cd zum neuen Ordner: (Version kann sich ändern)
   cd ventoy-1.0.99/
   ```
4. Führen Sie **`VentoyGUI.x86_64`** aus, indem Sie entweder darauf doppelklicken oder über das Terminal:
   ```sh
   # Es werden Administratorrechte angefordert, geben Sie Ihre Anmeldeinformationen ein
   ./VentoyGUI.x86_64
   ```
5. Wählen Sie das angeschlossene USB-Laufwerk aus, indem Sie auf **`Device`** klicken.
6. Klicken Sie auf **`Install`**.
7. Nachdem die Installation von Ventoy auf dem angeschlossenen USB-Laufwerk abgeschlossen ist, schließen Sie das Fenster und legen Sie die CachyOS-ISO-Datei in die neue nutzbare Partition des Laufwerks.

</Steps>

</TabItem>

<TabItem label="Windows">

<Steps>
1. Schließen Sie ein USB-Laufwerk an.
2. Laden Sie [Ventoy](<https://www.ventoy.net/en/download.html>) herunter.
3. Führen Sie **`Ventoy2Disk.exe`** aus.
4. Wählen Sie das angeschlossene USB-Laufwerk aus, indem Sie auf **`Device`** klicken.
5. Klicken Sie auf **`Install`**.
6. Nachdem die Installation von Ventoy auf dem angeschlossenen USB-Laufwerk abgeschlossen ist, schließen Sie das Fenster und legen Sie die CachyOS-ISO-Datei in die neue nutzbare Partition des Laufwerks.
</Steps>

</TabItem>

</Tabs>

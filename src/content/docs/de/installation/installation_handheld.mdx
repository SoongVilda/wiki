CachyOS bietet eine Edition für Handheld-Geräte wie Steam Deck, ROG Ally und Legion Go. Diese Edition bietet ein SteamOS-ähnliches Erlebnis mit Game Mode Switching, vorinstallierten Gaming-Anwendungen und mehr.

Die Handheld Edition verwendet den LAVD-Scheduler als Standard-CPU-Scheduler, der für Handheld-Geräte optimiert ist. Dies führt zu verbesserten Frameraten und einer längeren Akkulaufzeit beim Spielen.

Die Handheld Edition verwendet `systemd-boot` als Bootmanager. Die Auswahl des Bootmanagers ist nicht möglich,
im Gegensatz zur Standard-CachyOS-ISO. Dies soll den Installationsprozess vereinfachen.

## Installation auf Root

<Steps>

1. **Laden Sie** die neueste Handheld-ISO von der Website/dem Forum herunter.

2. [Flashen](/de/installation/installation_prepare) Sie die ISO.

3. **Booten** Sie in die ISO.

4. Klicken Sie auf die Schaltfläche **Launch Installer**.

5. Calamares wird nun geöffnet. Folgen Sie den Anweisungen auf dem Bildschirm.

6. Verwenden Sie beim Partitionierungsschritt bitte die Option **Festplatte löschen**. Wenn Sie von einer anderen Linux-Distribution kommen, die wahrscheinlich ein anderes Partitionierungslayout verwendet. Beachten Sie, dass die Option zum Ersetzen von Partitionen problematisch sein kann und möglicherweise nicht wie erwartet funktioniert.

7. Installieren Sie das System.

</Steps>

Nachdem die Installation abgeschlossen ist, werden Sie von Calamares aufgefordert, das Gerät neu zu starten.

Der erste Start kann etwas dauern, da Steam heruntergeladen und gestartet wird.
Dieser Vorgang kann bis zu 2 Minuten dauern.


## Installation mit Dual-Boot

<Steps>

1. **Laden Sie** die neueste Handheld-ISO von der Website/dem Forum herunter.

2. [Flashen](/de/installation/installation_prepare) Sie die ISO.

3. **Booten** Sie in die ISO.

4. Klicken Sie auf die Schaltfläche **Launch Installer**.

5. Calamares wird nun geöffnet. Folgen Sie den Anweisungen auf dem Bildschirm.

6. Beim Partitionierungsschritt müssen Sie **Manuelle Partitionierung** auswählen und folgende Partitionen erstellen:

    - 2 GB /boot
    - XGB / (X kann eine beliebige Menge an Speicherplatz sein, die Sie dem Root-Dateisystem zuweisen möchten)

7. Fahren Sie mit den Schritten fort und installieren Sie das System.

</Steps>

Nachdem die Installation abgeschlossen ist, werden Sie von Calamares aufgefordert, das Gerät neu zu starten.

Der erste Start kann etwas dauern, da Steam heruntergeladen und gestartet wird.
Dieser Vorgang kann bis zu 2 Minuten dauern.

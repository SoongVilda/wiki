## Pobieranie ISO

Obraz ISO CachyOS można pobrać z następujących źródeł:

*   [Strona internetowa](<https://cachyos.org/download>)
*   [SourceForge](<https://sourceforge.net/projects/cachyos-arch/files/>)
*   [Mirror CachyOS](<https://mirror.cachyos.org/ISO/>)

## Weryfikacja integralności ISO za pomocą SHA256

:::caution
Zawsze wykonaj dodatkowy krok weryfikacji integralności ISO, aby uniknąć niepożądanych problemów podczas instalacji lub tworzenia nośnika startowego.
:::

**Aktualna wersja ISO:** `250530`

**Skrót SHA256:** `7a24ed79c46e2ff4600e2c1b3f62883484cf21cae362f738e49025c5a2d957c9`

<Tabs>

<TabItem label='Windows'>

<Steps>

1.  Pobierz [plik](https://mirror.cachyos.org/ISO/desktop/250530/cachyos-desktop-linux-250530.iso.sha256) zawierający skrót SHA256 **(Otwórz go za pomocą edytora tekstu, np. Notatnika).**
2.  Otwórz Wiersz Polecenia (CMD) lub PowerShell jako Administrator i [przejdź](<https://www.wikihow.com/Change-Directories-in-Command-Prompt>) do ścieżki, w której zapisane są pliki ISO i SHA256.
3.  Wykonaj następujące polecenie:
    ```powershell
    # Przykład:
    certUtil -hashfile cachyos-desktop-linux-250530.iso SHA256
    ```
4.  Porównaj wynik polecenia certUtil ze skrótem z pobranego pliku w **Kroku 1.** Jeśli są zgodne, możesz kontynuować instalację CachyOS.

</Steps>

</TabItem>

<TabItem label='Linux'>

<Steps>

1.  Pobierz [plik](https://mirror.cachyos.org/ISO/desktop/250530/cachyos-desktop-linux-250530.iso.sha256) zawierający skrót SHA256.
2.  Otwórz terminal, przejdź do katalogu zawierającego plik `.sha256` i wykonaj następujące polecenia:
    ```sh
    # Przykład:
    cd Pobrane/ # Lub inna ścieżka
    cat cachyos-desktop-linux-250530.iso.sha256
    # 7a24ed79c46e2ff4600e2c1b3f62883484cf21cae362f738e49025c5a2d957c9
    ```
3.  Porównaj wynik z **Kroku 2** i wykonaj następujące polecenie, aby sprawdzić aktualny skrót pliku ISO.
    ```sh
    # Przykład:
    sha256sum cachyos-desktop-linux-250530.iso
    ```
4.  Jeśli skróty z **Kroku 2** i **Kroku 3** są zgodne, możesz kontynuować instalację CachyOS.

</Steps>

</TabItem>

## Weryfikacja autentyczności obrazu ISO (Linux)

Aby zweryfikować autentyczność pliku ISO i upewnić się, że został on wydany przez oficjalny zespół deweloperski CachyOS:

<Steps>

1.  Zaimportuj klucz GPG służący do weryfikacji autentyczności:

    ```shell
    gpg --keyserver hkps://keys.openpgp.org --recv-key F3B607488DB35A47
    ```

2.  Pobierz plik ISO oraz odpowiadający mu plik sygnatury `.sig`, a następnie uruchom następujące polecenie (zastępując `pełna_nazwa_iso.iso` rzeczywistą nazwą pliku ISO):
    ```shell
    gpg --verify pełna_nazwa_iso.iso.sig pełna_nazwa_iso.iso
    ```

    Jeśli otrzymasz wynik **Good signature** (Dobra sygnatura), plik ISO jest autentyczny:
    ```
    gpg: Signature made Mo 01 Apr 2024 14:13:30 CEST
    gpg:                using RSA key 882DCFE48E2051D48E2562ABF3B607488DB35A47
    gpg: Good signature from "CachyOS <admin@cachyos.org>" [unknown]
    gpg: WARNING: This key is not certified with a trusted signature!
    gpg:          There is no indication that the signature belongs to the owner.
    Primary key fingerprint: 882D CFE4 8E20 51D4 8E25  62AB F3B6 0748 8DB3 5A47
    ```

</Steps>

:::danger
Jeśli wynik nie zawiera ciągu **Good signature** (Dobra sygnatura) lub ID klucza się nie zgadza, nie używaj obrazu ISO i upewnij się, że pobrany obraz pochodzi z legalnego źródła CachyOS. Zła sygnatura może sugerować, że pobrany obraz został zmodyfikowany.
:::

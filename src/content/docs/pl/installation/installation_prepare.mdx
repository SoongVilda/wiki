## Wymagania systemowe

Przed rozpoczęciem przygotowań do instalacji upewnij się, że używany komputer spełnia niezbędne wymagania systemowe do uruchomienia CachyOS. Instalator
korzysta z procesu instalacji online, więc stabilne i stosunkowo szybkie połączenie internetowe jest obowiązkowe.

<Tabs>

<TabItem label="Wymagania minimalne">

*   3 GB RAM
*   30 GB przestrzeni dyskowej (HDD/SSD)
*   Stabilne połączenie internetowe

:::caution
Niestabilne połączenie internetowe może spowodować wydłużenie czasu instalacji lub, w najgorszym przypadku, niepowodzenie instalacji.
:::

</TabItem>

<TabItem label="Wymagania zalecane">

*   8 GB RAM
*   50 GB przestrzeni dyskowej (SSD/NVMe)
*   [Procesor zgodny z x86-64-v3](/pl/installation/installation_prepare#wsparcie-dla-poziomu-mikroarchitektury-x86_64)
*   Prędkość internetu 50 Mbps lub lepsza
*   Karta graficzna NVIDIA (seria 900+ - np. GTX 950), AMD +GCN 1.0 (np. AMD R7 240) lub Intel (zintegrowana seria HD Graphics lub nowsza. Seria Arc)

</TabItem>

</Tabs>

## Tworzenie rozruchowego dysku USB CachyOS

Istnieje kilka narzędzi i metod, których można użyć do utworzenia rozruchowego dysku USB CachyOS.

:::note
Dysk USB musi mieć co najmniej 8 GB wolnego miejsca.
:::

### balenaEtcher

<Steps>

1. Pobierz [balenaEtcher](<https://etcher.balena.io/>). balenaEtcher obsługuje flashowanie obrazu w systemach Linux, MacOS i Windows.
   Jeśli pobrano plik `.zip`, rozpakuj go i uruchom plik binarny za pomocą następujących poleceń:
      ```sh
      # Rozpakuj plik zip:
      bsdtar xvf <nazwa pliku zip>
      # Przejdź do nowego folderu:
      cd balenaEtcher-linux-x64/
      # Uruchom balenaEtcher:
      ./balena-etcher
      ```
2. Podłącz dysk USB do komputera.
3. Wybierz **`Flash z pliku`** i wskaż plik obrazu ISO CachyOS.
4. Kliknij **`Wybierz cel`** i wybierz wcześniej podłączony dysk USB.
5. Naciśnij **`Flashuj`**, aby rozpocząć proces flashowania.

</Steps>

:::note
balenaEtcher może poprosić o uprawnienia administratora. Jeśli tak, wprowadź hasło i kliknij **`Uwierzytelnij`**.
:::

### Wiersz poleceń (Linux i MacOS)

<Steps>

1. Podłącz dysk USB.
2. Wykryj podłączony dysk USB za pomocą następujących poleceń:
   ```sh
   # Linux
   # Dysk USB można zidentyfikować np. jako /dev/sda, na podstawie modelu i rozmiaru dysku.
   sudo fdisk -l

   # MacOS
   # Dyski USB są identyfikowane jako "/dev/diskY", gdzie "Y" może być 0, 1 itd.
   diskutil list
   ```
3. Skopiuj zawartość ISO na podłączony dysk USB, uruchamiając:
   ```sh
   # Zastąp <usbdrive> etykietą dysku.
   sudo dd bs=4M if=pelna_nazwa_iso.iso of=/dev/<usbdrive> status=progress oflag=sync
   ```
</Steps>

- `dd` skopiuje teraz zawartość pliku ISO na podłączony dysk USB.

### Rufus (Tylko Windows)

<Steps>

1. Podłącz dysk USB.
2. Pobierz [Rufus](<https://rufus.ie/>) i zainstaluj go lub uruchom wersję przenośną.
3. W polu **`Urządzenie`** kliknij listę rozwijaną i wybierz podłączony dysk USB.
4. W polu **`Wybór rozruchu`** kliknij **`WYBIERZ`** i zlokalizuj plik obrazu ISO CachyOS.
5. Kliknij **`START`**.

</Steps>

### Ventoy (Linux i Windows)

<Tabs>

<TabItem label="Linux">

<Steps>

1. Podłącz dysk USB.
2. Pobierz [Ventoy](<https://www.ventoy.net/en/download.html>)
3. Rozpakuj Ventoy za pomocą następującego polecenia:
   ```sh
   # Zastąp wersję poprawną, np.: ventoy-1.0.99-linux.tar.gz
   bsdtar xvf ventoy-wersja-linux.tar.gz
   # Przejdź do nowego folderu: (wersja może się zmienić)
   cd ventoy-1.0.99/
   ```
4. Uruchom **`VentoyGUI.x86_64`** poprzez dwukrotne kliknięcie lub przez terminal:
   ```sh
   # Zostaniesz poproszony o uprawnienia administracyjne, wprowadź swoje dane uwierzytelniające
   ./VentoyGUI.x86_64
   ```
5. Wybierz podłączony dysk USB, klikając **`Urządzenie`.**
6. Kliknij **`Zainstaluj`.**
7. Po zakończeniu instalacji Ventoy na podłączonym dysku USB zamknij okno i umieść plik ISO CachyOS na nowej, użytecznej partycji dysku.

</Steps>

</TabItem>

<TabItem label="Windows">

<Steps>
1. Podłącz dysk USB.
2. Pobierz [Ventoy](<https://www.ventoy.net/en/download.html>)
3. Uruchom **`Ventoy2Disk.exe`**
4. Wybierz podłączony dysk USB, klikając **`Urządzenie`.**
5. Kliknij **`Zainstaluj`.**
6. Po zakończeniu instalacji Ventoy na podłączonym dysku USB zamknij okno i umieść plik ISO CachyOS na nowej, użytecznej partycji dysku.
</Steps>

</TabItem>

</Tabs>
